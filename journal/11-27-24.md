---
title: Functionality Design
date: 2023-11-27
---

- Uploading of music audio file
- Vocal extraction (using UVR)
- Detection of key and tempo
- Converts both instrumental and vocal tracks to notes
- Displays note names of the current scale at the left of the track
- Displays chord progression at bottom of the track
- Automatically scrolls to the left as the music plays
- User vocal input and pitch detection
- User playing the virtual instrument using keyboard or touch screen
- Real-time scoring of pitch accuracy
- Lyrics editor with timestamps
- Clicking and dragging to edit notes
- Vocal synthesis (no lyrics, only notes for simplification)
- Chord progression editor
- Rhythmic pattern editor
- Generate instrumental track according to chord progression and rhythmic pattern 
- Pattern library and presets (pop, rock, jazz, funk, blues, etc.)
- Loop section markers
